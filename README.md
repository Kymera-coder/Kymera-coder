# 🚀 Kymera Python Coder Studio 4.19.3 — Editor AI Pro

## ✨ Visão Geral

O **Kymera Coder** é uma IDE moderna com integração de IA, inspirada no VS Code, projetada para desenvolvedores que querem produtividade máxima com assistência inteligente integrada.

Com suporte a execução de código, terminal embutido, sistema de IA em tempo real e ferramentas avançadas de edição, o Kymera oferece uma experiência completa em um único ambiente focada em Python.

---

## 🔥 Principais Recursos

### 🧠 Integração com IA (Copilot)

* Chat inteligente integrado
* Geração de código em tempo real
* Correção automática de bugs
* Explicação de código
* Formatação e otimização (PEP8)

### 🧩 Editor Avançado

* Destaque de sintaxe (Python)
* Numeração de linhas
* Autocompletar inteligente
* Atalhos:

  * `Ctrl + D` → duplicar linha
  * `Ctrl + /` → comentar/descomentar
* Drag & Drop de arquivos

### 📁 Explorador de Arquivos

* Navegação completa do sistema
* Abertura rápida de arquivos
* Integração com diretórios locais

### 🖥️ Terminal Integrado

* Execução de comandos em tempo real
* Navegação com `cd`
* Limpeza com `clear` / `cls`
* Execução de scripts Python diretamente

### 🛠️ Ferramentas Profissionais

* Localizar e substituir
* Ir para linha específica
* Alternar quebra de linha
* Duplicação de código
* Sistema de problemas (erros detectados)

### ⚙️ Execução e Build

* Execução rápida de scripts Python
* Compilação para `.exe` com PyInstaller
* Suporte a ícones personalizados

---

## 🎨 Interface

* Tema Dark inspirado no VS Code
* Layout dividido:

  * Explorer (esquerda)
  * Editor + Terminal (centro)
  * IA Copilot (direita)
* UI moderna com PyQt6

---

## 📦 Tecnologias Utilizadas

* Python 3.10+
* PyQt6
* Google Generative AI (Gemini)
* PyInstaller

---

## ⚙️ Instalação e Execução

### 📥 1. Baixe o projeto

* Faça o download do arquivo `.py` diretamente do repositório
  ou clone:

```bash
git clone https://github.com/kymera-coder/kymera-coder.git
```

---

### 🐍 2. Instale as dependências

```bash
pip install PyQt6 google-generativeai pyinstaller
```

---

### ▶️ 3. Execute o Kymera

```bash
KPC Studio 4.19.3.py
```

> ⚠️ Certifique-se de estar usando Python 3.10 ou superior

---

### 💡 Dica

Se preferir, você pode executar o arquivo dando dois cliques no `.py`, desde que o Python esteja configurado no sistema.

[KPC Studio 4.19.3.PY](https://github.com/user-attachments/files/26318832/Kc.Studio.4.19.3.PY)


## 🔑 Configuração da IA

1. Obtenha sua API Key do Gemini
2. Insira no campo superior da aplicação
3. Clique em **Conectar**
